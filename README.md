**サービス名**
TestScoreRivals

**サービス概要:**
このアプリは、友達やクラスメイトとテストの点数を競争するためのプラットフォームです。ホストが問題を作成し、ユーザーがその問題を解くことで、点数化、順位化を簡潔に行うことができます。また、ユーザーが問題を作成することも可能で、作成した問題を共有することもできます。ユーザーはテストのスコアを確認し、友達とのスコア比較やランキングを見ることができます。また、学習目標の設定と共有もサポートします。
テストの点数や順位を自分を客観視することで、自分の現在の立ち位置を知ることができます。
また、不正に関しては、
セキュリティと不正防止:問題が不正に改ざんされたり、解答がコピーされたりするのを防ぐためにセキュリティ対策を実施します。
問題のプライバシー設定:ホストが作成した問題の公開範囲を設定できるようにし、一部の問題をプライベートに保つオプションを提供します。
信頼性スコア: ユーザーの信頼性スコアを導入することで、信頼性の低いユーザーによる不正行為を軽減できます。ユーザーの過去の行動や成績に基づいてスコアを計算し、信頼性の高いユーザーがランキングに影響を与えるようにします。

**このサービスへの思い・作りたい理由:**
学習に対するモチベーションは競争から生まれることがあり、友達との競争を通じて学習を楽しみ、向上するためのプラットフォームが必要と考えています。
学生時代、私は意志が弱く、勉学に対してのモチベーションを保つのがとても難しかったです。順位が出る定期考査や模試などは頻繁に行われるものではなく、自分の立ち位置を客観的に知ることができませんでした。そして、気付いた時には手遅れといった感じでした。また、比較される順位はクラスや学年全体で志望校も偏差値も異なるメンバーです。自分とレベルの近いライバルとグループを作って比較することで、自分の競争心やモチベーション上げるきっかけになるのではないかと考えました。ライバルの存在は自分にとってとても力になるものです。
また、現在の職業から、生徒が回答した問題を丸つけし、パソコンにデータを打ち込み、そのデータで順位分けをするという作業はとても面倒です。この作業をひとつにできるアプリはできないかと、開発しました。


**ユーザー層:**
このアプリの主なユーザー層は次のようなグループを対象にしています：
高校生向け競争プラットフォーム：高校生をターゲットにし、高校の生徒がテストの成績を競い合うプラットフォームとして位置づけることができます。高校生はテストなどが多く、また、競争心をもつことでモチベーションの向上につながります。このプラットフォームを使用して友達やクラスメイトと、塾のメンバーなどと意識高く勉強に取り組めると思います。
成績の近い人同士や、志望校のレベルが近い人とグループを組むのも良いです。
共通の趣味を持っている仲間：自分で問題を作成できるので、それを共有し合うことで知識勝負のような感じで楽しむこともできます。

**サービスの利用イメージ:**
ユーザーはこのアプリを使用して、以下の価値を得ることができます：
- 友達やクラスメイトとのテストスコアの競争
- 学習目標の設定と共有
- 問題作成、問題共有を行うことで、知識比べができる

**ユーザーの獲得について:**
ユーザー獲得のために、以下の方法を検討しています：
- 学生向けの学習コミュニティやオンラインフォーラムでアプリを宣伝。
- ソーシャルメディアプラットフォームで友達との競争を奨励。
- 学校や大学と提携し、教育機関内での導入を検討。

**サービスの差別化ポイント・推しポイント:**
このアプリの差別化ポイントは以下の点です：
- 問題作成、点数化、順位分けの作業を一括で行うことができる。
- 友達やクラスメイトとの競争機能が強調されており、学習モチベーションを高める。
- 学習目標の共有と共感をサポート。
- 目標達成の喜び: 学習目標を設定し、達成することは充実感と喜びをもたらします。ユーザーが目標を達成するたびに、成功体験を共有できるプラットフォームがユーザーに提供する売りポイントです。
- 問題カテゴリ化:問題を主題、難易度、科目などに基づいてカテゴリ化し、ユーザーが特定の問題を見つけやすくします。
- 競争と協力: 学習目標の共有は競争心を高めるだけでなく、協力と共感も促進します。友達やクラスメイトと共に目標を設定し、達成することで、学習をより豊かな体験に変えることができます。

**機能候補:**
MVPリリース時に含める機能：
- フレンドリストとチャット機能: ユーザーが友達やクラスメイトをフレンドリストに追加し、彼らとリアルタイムでチャットできる機能。これにより、ユーザー同士のコミュニケーションを促進し、競争や学習目標の共有がスムーズに行えます。
- 報酬とリーダーボード: ユーザーに達成感を提供するための報酬システムと、トップパフォーマーを表示するリーダーボード。ランキング形式の競争がモチベーション向上に貢献します。
- 問題のインポート/エクスポート: ユーザーが外部から問題を取り込んだり、自分が作成した問題をエクスポートしたりできる機能。これにより、問題の多様性とシェアビリティが向上し、ユーザー間の交流が促進されます。
- 通知とリマインダー: テストや学習目標の期限に近づくと、ユーザーに通知やリマインダーを送信する機能。これはユーザーがスケジュールを守り、目標を達成するのに役立ちます。また、ホストやユーザーが問題を作成した時に通知をしてくれます。
- 学習目標の設定と追跡
 共感とサポート: 学習目標を共有し、友達やクラスメイトと共感し合い、サポートし合うことができます。
 ユーザーがダッシュボードにアクセスし、学習目標を設定します。例えば、次回のテストで得点を10点上げるという目標を設定できます。ダッシュボードにはこの目標が表示され、進捗バーで進行状況が確認できます。
- 問題のカスタマイズ: ユーザーが問題を自由にカスタマイズできるエディター機能。これにより、問題の多様性とユーザーエンゲージメントが向上します。

本リリースまでに作成したい機能：
- タスク管理機能
- プッシュ通知
- 課題提出機能
- グループ間でのチャット機能
- 学習記録と進捗レポート
- グループ競争機能
- 報酬とトロフィー
- ホスト、ユーザーの問題作成機能
- 問題の種類や難易度のタグ付けやカテゴリ分け機能

**機能の実装方針予定:**
- フレンドリストとチャット機能:チャット機能の実装には、リアルタイム通信のためのWebSocketを使用します。一般的なWebSocketライブラリやフレームワークを選んで実装し、ユーザー間のリアルタイムチャットを実現します。また、ユーザーのフレンドリストを管理するためにデータベースを使用し、友達の追加や削除、メッセージの保存などを処理します。
- 問題のインポート/エクスポート:問題のインポートには、CSVファイルやJSONファイルのアップロードをサポートするAPIを使用します。ファイルアップロード機能を実装し、アップロードされたファイルをパースして問題データをデータベースに保存します。問題のエクスポートには、データをCSVやJSON形式でエクスポートするエンドポイントを提供します。
- グループ競争機能：友達をグループに招待し、グループ内での競争をサポートします。
-ポイントとランキング: グループメンバーのスコアをリアルタイムで追跡し、ランキングを生成します。これにはデータベース（例：Firebase、MongoDB）を使用して、ポイントを管理する方法を使います。
- グループの目標設定: グループメンバーが共通の学習目標を設定できる機能を提供します。
- リレーショナルデータベース（例：MySQL）、NoSQLデータベース（例：MongoDB）、またはクラウドデータベース（例：Firebase Realtime Database、Firestore）を使用します。
- 問題のカスタマイズ:問題のカスタマイズには、問題エディターを実装します。エディターで問題を編集し、保存できるようにします。問題の種類、難易度、カテゴリなどのタグを問題に追加し、問題の検索やフィルタリングを容易にします。問題データはデータベースに格納され、ユーザー間で共有できるようになります。
- データの暗号化:問題とユーザーデータを保存および転送する際に、適切な暗号化を実施します。これにより、データが盗まれたり改ざんされたりするリスクが軽減されます。
- プライバシー設定:ホストが作成した問題に対するプライバシー設定を提供し、一部の問題をプライベートに保つオプションを提供します。これにより、ユーザーは問題の公開範囲を選択できます。
